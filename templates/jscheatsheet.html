General Differences from Python

  • All statements must end with a semicolon
  • Use // instead of # for comments
  • Use console.log('hello') instead of print 'hello'
  • The first (and only the first!) time we assign to a variable we have to write var before it to declare the variable
  • Use === and !== for equality comparisons instead of == and !=
  • Use { and } to define code blocks instead of colons and indenting
  • Use else if instead of elif
  • true and false are lowercase instead of uppercase
  • Use var foo = function(arg1, arg2) { instead of def foo(arg1, arg2):
  • We say 'array' instead of 'list' and 'object' instead of 'dictionary'

 

Variables

Assigning to a variable

Note: The first time we assign to a variable we have to declare the variable with the 'var' keyword. 
> var a_string = "hello world";
> console.log(a_string);
hello world

> var a_num = 23;
> console.log(a_num);
23

> var a_string = "hello world";
> console.log(a_string);
> a_string = "goodbye world";
> console.log(a_string);
hello world
goodbye world

Using a variable in place of a number or a string

> 5 + 6;
11

> var x = 5;
> console.log(x + 6);
11

> var a_string = "hello"
> console.log(a_string + "world");
helloworld

 

If Statements

Note: When checking for equality use triple === instead of the single = we use for assignment
5 === 5 evalulates to true since 5 equals 5
5 === 6 evaluates to false since 5 does not equal 6
x = 5 assigns 5 to x a single = sign means assignment
5 !== 6 evaluates to true since 5 does not equal to 6

Basic If Statement

> if (5 === 5) {
>    console.log('hello');
> }
hello

> if (6 === 5) {
>    console.log('hello');
> }

Basic If/Else Statement

> if (6 === 5) {
>   console.log('hello');
> }


> if (5 === 5) {
>    console.log('5 equals 5');
> } else {
>    console.log('5 not equal to 5');
> }
5 equals 5

> if (5 === 6) {
>    console.log('5 equals 6');
> } else {
>    console.log('5 not equal to 6');
> }
5 not equal to 6

If, Else If, Else If, Else

Checks each if, else if statement in order until finds one that's true and executes the body and skips the rest of else if/else cases. If none of the if/else if are satisfied, the body of the else executes. 
> if (5 > 6) {
>    console.log('5 greater than 6');
> } else if (5 === 6) {
>    console.log('5 equal to 6');
> } else if (5 < 6) {
>    console.log('5 less than 6');
> } else {
>    console.log("it's all a lie");
> }
5 less than 6

If, If, If vs If, Else if, Else if

Let's see what happens in each case by walking through the code.
If you use three if statements, they run independently from each other. If the first if is true, then we evaluate the body of the first if statement and check if the second if statement is true. If the second if statement is true we evaluate the body of the second if statement and then check if the third if is true. If the third if is true, we evaluate the body of the third if statement.
On the other hand, in the If/Else if/Else if case, we first check if the first if is true. If the first if is true, then we evaluate the body of the first if statement and then we complete the evaluation of the entire if/else if/else if block. 
> if ('hello'.indexOf('h') !== -1) {
>    console.log('h is in hello');
> }
> if ('hello'.indexOf('e') !== -1) {
>    console.log('e is in hello');
> }
> if ('hello'.indexOf('l') !== -1) {
>    console.log('l is in hello');
> }
h is in hello
e is in hello
l in hello

> if ('hello'.indexOf('h') !== -1) {
>    console.log('h is in hello');
> } else if ('hello'.indexOf('e') !== -1) {
>    console.log('e is in hello');
> } else if ('hello'.indexOf('l') !== -1) {
>    console.log('l is in hello');
> }
h is in hello

 

Arrays

Creating an array

> var an_array = []; // creates an empty array
> console.log(an_array);
[]

> var an_array = [1, 2, 3, 4]; // creates an array with numbers already inside
> console.log(an_array);
[1, 2, 3, 4]

Adding elements to end of array

> var an_array = [];
> an_array.push(5);
> an_array.push(4);
> an_array.push(3);
> an_array.push(2);
> an_array.push(1);
> an_array.push("BLAST OFF!");
> console.log(an_array);
[5, 4, 3, 2, 1, 'BLAST OFF!']

Getting parts of an array

> var an_array = ['hello', 'this', 'is', 'an', 'array'];
> console.log(an_array[0]); // prints the first element of an_array
> console.log(an_array[1]); // prints the second element of an_array
> console.log(an_array[2]); // prints the third element of an_array
> console.log(an_array.slice(0, 2)); // prints the first up to the third element of an_array (not including third element)
> console.log(an_array.slice(2)); // prints all the elements from the third element of the array to the end of the list
> console.log(an_array.slice(2, 4)); // prints the third up to the fifth element of an_array (not including the fifth element)
hello
this
is
['hello', 'this']
['is', 'a', 'list']
['is', 'a']

Checking if an element is inside array

> var array_of_ingredients = ['apples', 'strawberries', 'bananas', 'kiwis'];
> console.log(array_of_ingredientes.indexOf('pineapples'));
-1

> var array_of_ingredients = ['apples', 'strawberries', 'bananas', 'kiwis'];
> console.log(array_of_ingredients.indexOf('apples'));
> console.log(array_of_ingredients.indexOf('bananas'));
0
2

Functions

Creating a function with no arguments

> var return_10 = function() {
>    return 10;
> };
> console.log(return_10());
10

Creating a function with one argument

> var multiply_by_10 = function(x) {
>    return 10 * x;
> };
> console.log(multiply_by_10(10));
> console.log(multiply_by_10(5));
100
50

Creating a function with multiple arguments

> var return_array_multiplied_by_x = function(an_array, x) {
>    var new_array = [];
>    for (var i = 0; i < an_array.length; i++) {
>        new_array.push(an_array[i] * x);
>    }
>    return new_array;
> };
> console.log(return_array_multiplied_by_x([1,2,3,4,5], 5));
> console.log(return_array_multiplied_by_x([1,2,3,4,5], 3));
> console.log(return_array_multiplied_by_x([1,2,3,4,5], 2));
[5, 10, 15, 20, 25]
[3, 6, 9, 12, 15]
[2, 4, 6, 8, 10]
