FAQ

How much programming experience should I have coming into this course? None. Nada. Zip. Zero. The course is designed to teach you from the ground up!
Do I need a strong math background to take this DeCal? If you can do arithmetic, you're good to go.
Will there be homework? Yes but we don't like homework just as much as you.
Will the lectures and readings be available online? The readings are all online and we will do our best to webcast lectures so you can review.
What will I actually learn from this course? You will learn Javascript, the scripting language of the internet. You'll start by building a personal website, and learn how to use Javascript to add interactivity. About halfway through the course, we'll dive into some real world applications such as games, web scraping, data processing, and building a full web application.
Will you teach me to make my website beautiful? Unfortunately, no. Though we'll touch on web development, we will NOT be teaching you how to make a nice-looking website, aka web design. For that, the folks over at The Web Design Workshop seem to know what they're doing.
Is attendance mandatory? Yes
Did anybody actually ask any of these questions? Nope
